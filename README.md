# 🎓 «Корочки.есть» — система записи на курсы ДПО

## О проекте

Сервис предназначен для автоматизации записи слушателей на программы дополнительного профессионального образования. Пользователи могут самостоятельно выбирать курсы, подавать заявки, отслеживать их статус и оставлять отзывы. Администратор имеет возможность управлять заявками и менять их состояние.

## ✨ Что умеет система

### Для пользователя
- Создание учётной записи и вход в личный кабинет
- Просмотр доступных курсов
- Заполнение и отправка заявки на обучение
- Отслеживание статуса заявки (новая / в обработке / одобрена / отклонена)
- Написание отзыва после завершения курса

### Для администратора
- Просмотр всех поступивших заявок
- Изменение статуса каждой заявки
- Контроль активности пользователей

## 🔧 Как это работает

- Данные отправляются и загружаются без перезагрузки страницы (Fetch)
- Пароли пользователей сохраняются в зашифрованном виде (bcrypt)
- Все входные данные проходят проверку — и в браузере, и на сервере
